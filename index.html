About
Barad is a multipurpose bot built for security, moderation, and promoting transparency between moderation and server members, with a lot of other fun commands on the side!
Use this bot for:
  • Server Defense
  • Mass Edits
  • Meme Generation
  • and more!
Supports slash commands, mentions, and a prefix ([])
Use the nav menu to explore our
commands and functions!

Moderation Commands

Commands only available to server members with specific permissions
A majority of these commands were developed to easily utilized moderation function or to easily enact mass edits on large numbers of users

Note: These commands can be easily abused, so be careful with who has permissions to use them
mass rename

Renames all selected users


Parameters:

(required) targets
mentions of users to be renamed (mention roles to select all members with those roles)
(optional) exclude
mentions of users and roles to be excluded from renaming (default: none)
(optional) setting
whether to set names or reset to default (default: 'Set')
(optional) name
nickname to set for selected users if setting is 'Set' (names will be set to gibberish of setting is 'Set' and no name is provided)
(optional) reason
reason for renaming to be added in audit logs (default: none)

User Permissions: manage nicknames


Only users under command author and bot in roles will be edited

Bot Permissions: manage nicknames, send messages


Only useable in servers
mass rolemod

Edits roles of all selected users


Parameters:

(required) roles
mentions of roles to be edited for selected users (only roles under bot can be selected)
(required) targets
mentions of users to have their roles edited (mention roles to select all members with those roles)
(optional) exclude
mentions of users and roles to be excluded from role editing (default: none)
(optional) setting
whether to give the roles, take the roles, or set the roles (default: 'Give') Note: setting will add the roles but remove all other roles from a user and can only be done for users under command author and bot in roles
(optional) reason
reason for role edit to be added in audit logs (default: none)

User Permissions: manage roles


Command author must be above bot in roles

Bot Permissions: manage roles, send messages


Only useable in servers
mass kick

Kicks all selected users


Parameters:

(required) targets
mentions of users to be kicked (mention roles to select all members with those roles)
(optional) exclude
mentions of users and roles to be excluded from kicking (default: none)
(optional) reason
reason for kick to be added in audit logs (default: none)

User Permissions: kick members


Command author must be above bot in roles

Only members under command author will be kicked

Bot Permissions: kick members, send messages


Only useable in servers
mass ban

Bans all selected users


Parameters:

(required) targets
mentions of users to be banned (mention roles to select all members with those roles)
(optional) exclude

(optional) reason
reason for ban to be added in audit logs (default: none)

User Permissions: ban members


Command author must be above bot in roles

Bot Permissions: ban members, send messages

purge

Removes elements containing a certain key


Parameters:

(required) key
string to search for in server
(optional) purge_channels
whether to remove channels with names containing key (default: 'No')
(optional) purge_roles
whether to remove roles with names containing key (default: 'No')
(optional) purge_emojis
whether to remove emojis with names containing key (default: 'No')
(optional) purge_users
whether to remove users with names containing key (default: 'No')
(optional) reason
reason for purge to be added in audit logs (default: none)
Must select 'Yes' for at least one purge option

User Permissions: administrator


Command author must be above bot in roles

Bot Permissions: send messages, all permissions to delete selected items


Only useable in servers

Automod Functions

Automatic actions taken by bot under certain conditions

Note: These cannot be disabled and apply to any users below bot in roles
Kick/Ban Logger

Any server member below bot in roles is banned if they remove more than 10 users in a period of 5 minutes or less.


Bot Permissions: ban members


Only useable in servers
Channel Deletion Logger Logger

Any server member below bot in roles is banned if they delete more than 10 channels in a period of 5 minutes or less.


Bot Permissions: ban members


Only useable in servers
Role Deletion Logger

Any server member below bot in roles is banned if they delete more than 10 channels in a period of 5 minutes or less.


Bot Permissions: ban members


Only useable in servers
Anti-Spam Moderation

Any message that has already been repeated more than 5 times within 30 seconds in any channel not containing "spam" in its name will be deleted.


Bot Permissions: manage messages


Only useable in servers

Discord Roulette

A fun game to play with friends!

To set up a server for the game the following channels and roles need to be created:

Roles Channels
@Playing #player-chat
@Killer #killer-chat
@Dead #dead-chat
@Spectator #registration

Note: Bot must be above all four roles and have access to all four channels
roulette info

Provides a link to this page.


Parameters: none


User Permissions: none


Bot Permissions: send messages


Only useable in servers
roulette register

Enters user into the game, gives them @Playing, and removes @Spectator


Parameters: none


User Permissions: none


Command author must have @Spectator role to register

Bot Permissions: manage roles, send messages


Only useable in servers
roulette unregister

Cancels registration by giving user @Spectator and removing @Playing


Parameters: none


User Permissions: none


Command author must have @Playing role to register

Bot Permissions: manage roles, send messages


Only useable in servers

Selects a random player to be killer and gives them @Killer


Parameters: none

roulette shoot

Simulates the shooting of a target, gives them @Dead if they are shot or makes them the next killer if they survive.


Parameters:

(required) targets
mention of a player to target
(required) bullets
number of bullets (1-6) to be loaded


User Permissions: none

Bot Permissions: manage roles, send messages


Only useable in servers

Parameters: none


User Permissions: none

Bot Permissions: manage roles, send messages


Only useable in servers
